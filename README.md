# 🧮 Python Calculator GUI

This is a simple Calculator GUI application built using Python and Tkinter.

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear button
- User-friendly interface

## 🛠️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run
1. Install Python
2. Download this project
3. Run the file:
   python calculator-gui.py
